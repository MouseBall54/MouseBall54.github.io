## 핵심 요약

Pomodoro는 시작이 어렵거나, 미루는 습관을 줄이거나, 작은 작업을 처리할 때 좋습니다.
Deep work는 읽기, 글쓰기, 코딩, 문제 풀이, 시험 준비처럼 긴 집중이 필요한 작업에 좋습니다.
둘 중 하나만 고를 필요는 없습니다.
Pomodoro로 시작하고, deep work로 중요한 시간을 보호하면 됩니다.

![Pomodoro와 deep work의 집중 session과 break를 보여주는 timeline](/images/2026-05-23-pomodoro-deep-work/pomodoro-deep-work-hero.png)

이미지는 timed work block, short break, longer review block이 섞인 구조를 보여줍니다.
목표는 모든 시간을 꽉 채우는 것이 아닙니다.
집중을 시작하기 쉽게 만들고, 유지하기 쉽게 만드는 것입니다.

## Pomodoro를 1분 안에 이해하기

Classic Pomodoro Technique은 timer, 집중 work interval, short break, 반복 cycle을 사용합니다.
많은 사람이 25분 집중과 5분 휴식을 사용합니다.
몇 cycle 뒤에는 더 긴 휴식을 둡니다.

이 방식이 좋은 이유는 시작 비용을 낮추기 때문입니다.
4시간 공부하겠다고 약속하는 것이 아니라, focused block 하나만 하겠다고 정하는 것입니다.

Pomodoro에 잘 맞는 작업:

- flashcard review
- 짧은 practice problem
- note 정리
- rough outline 작성
- email 또는 admin work
- 작은 error 하나 debug

Pomodoro에 덜 맞는 작업:

- warming-up이 긴 작업
- 자주 끊기면 흐름이 깨지는 writing
- timer가 오히려 집중을 끊는 deep reading

## Deep Work를 1분 안에 이해하기

Deep work는 인지적으로 어려운 작업을 방해 없이 지속하는 것입니다.
더 긴 block, 적은 interruption, 명확한 target이 필요합니다.

Deep work에 잘 맞는 작업:

- 긴 글 작성
- 어려운 수학 문제 풀이
- feature 구현
- 밀도 높은 chapter 읽기
- study plan 설계
- research paper review

Deep work는 단순히 오래 일하는 것이 아닙니다.
보호된 attention입니다.

## 실전 Hybrid Schedule

둘을 함께 씁니다.

```text
Warm-up: Pomodoro 1개
Main work: 60-90분 deep work block
Break: 10-20분
Cleanup: Pomodoro 1개
Review: 5분
```

공부 session 예시:

| Phase | Time | Goal |
| --- | ---: | --- |
| Pomodoro 1 | 25분 | 어제 약한 질문 review |
| Break | 5분 | 일어서기, 물 마시기, phone scroll 금지 |
| Deep work | 75분 | 새 문제 풀기 또는 기억에서 쓰기 |
| Break | 15분 | attention reset |
| Pomodoro 2 | 25분 | mistake note 만들기 |
| Review | 5분 | 다음 review 예약 |

이 구조는 낮은 마찰로 시작하게 해주고, 가장 중요한 block을 보호합니다.

## Block Length 고르는 법

짧은 block이 좋은 경우:

- 지루하지만 필요한 작업
- 피곤한 상태
- 주변 환경이 시끄러운 경우
- 긴 휴식 뒤 다시 시작하는 경우
- 작은 단계가 많은 작업

긴 block이 좋은 경우:

- context가 필요한 작업
- interruption 비용이 큰 작업
- 쓰기, 추론, 설계가 필요한 작업
- 무엇을 할지 이미 정해진 경우
- 시간을 보호할 수 있는 경우

다른 사람의 schedule을 그대로 복사하지 않습니다.
가장 좋은 schedule은 수면, 운동, 식사, 회복을 망치지 않고 유지할 수 있는 schedule입니다.

## Break 규칙

Break는 작은 work session이 아닙니다.
Attention을 reset하는 시간입니다.

좋은 break:

- 걷기
- stretching
- 물 마시기
- 화면 밖 보기
- desk 정리
- 1분간 천천히 호흡

위험한 break:

- social media
- short video
- news feed
- 새로운 어려운 작업 시작
- 급한 마음을 만드는 message 확인

Break 후 돌아오기 더 어려워진다면 그 break는 session을 돕지 못합니다.

## Focus Block Template

아래를 복사해 사용할 수 있습니다.

```text
Session goal:
Start time:
Block type: Pomodoro / Deep work
One task:
Definition of done:
Distractions to block:
Break plan:
Review note:
```

예시:

```text
Session goal: spaced repetition schedule 이해하기.
Block type: Deep work
One task: 기억에서 7일 review plan 작성.
Definition of done: table 1개와 weak point 3개.
Distractions to block: phone, email, unrelated browser tabs.
Break plan: 10분 걷기.
Review note: 약한 card를 내일 list에 추가.
```

## 흔한 실수

- Pomodoro를 어려운 작업을 피하는 도구로 씁니다.
- 90분이 필요한 작업에도 무조건 25분 block을 강요합니다.
- Message를 확인하면서 긴 session을 deep work라고 부릅니다.
- 집중이 무너질 때까지 break를 건너뜁니다.
- 시간을 기록하지만 output은 보지 않습니다.
- 이미 잠이 부족한데 focus block을 10개 계획합니다.

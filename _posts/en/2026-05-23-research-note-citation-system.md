## Quick Summary

Citation management starts while reading, not at the final bibliography stage.

This routine is not decoration for a longer study session. It should leave **source link** and **paraphrase** so the next session can decide what to repeat and what to reduce. Start with one subject and one unit before scaling it across a full schedule.

## Practical Routine

- Save the source link and access date.
- Rewrite the claim in your own words.
- Mark direct quotations with quotation marks.

## 40-Minute Session Example

If you only have 40 minutes today, start with 'Save the source link and access date'. Then record the **source link** result and separate correct items from confused items. Use the final five minutes to write one question that starts the next review. That small closing record prevents the next session from becoming setup time again.

## Record Example

The record does not need to be long. Filling three fields, **source link**, **paraphrase**, and **quote mark**, is enough for one session. Move correct items to a longer interval, tag confused items with a short reason, and put missed items at the top of the next session. This keeps the next study block from starting with setup work.

## Checklist

- Before starting, define the **source link** output for today.
- Before ending, check **paraphrase** and mark the next review item.
- Keep time spent, correct items, and missed items in one table.
- If the routine is too complex, remove one step and compare again next week.

## FAQ

### Should I apply Research Note and Citation System: Summarize Before Quoting to every subject immediately?

Start with one subject, one unit, and one review cycle. Expand **Research Note and Citation System: Summarize Before Quoting** only after the **source link** record is useful in the next session.

### Can this work when study time is short?

Yes, if the short session still checks **paraphrase** and leaves a closing record. In **Research Note and Citation System: Summarize Before Quoting**, time alone is not the point; retrieval, feedback, and rescheduling need to be included.

### Is Research Note and Citation System: Summarize Before Quoting failing if scores do not improve immediately?

No. **Research Note and Citation System: Summarize Before Quoting** first becomes valuable by revealing repeated failure points. Keep the same **source link** measure for two or three weeks before changing the system.

## Quick Summary

A score alone does not tell the next action; each item must become a study decision.

This routine is not decoration for a longer study session. It should leave **score section** and **guessed item** so the next session can decide what to repeat and what to reduce. Start with one subject and one unit before scaling it across a full schedule.

## Practical Routine

- Mark wrong and guessed items separately.
- Record where time pressure started.
- Schedule the retest date.

## 40-Minute Session Example

If you only have 40 minutes today, start with 'Mark wrong and guessed items separately'. Then record the **score section** result and separate correct items from confused items. Use the final five minutes to write one question that starts the next review. That small closing record prevents the next session from becoming setup time again.

## Record Example

The record does not need to be long. Filling three fields, **score section**, **guessed item**, and **time pressure**, is enough for one session. Move correct items to a longer interval, tag confused items with a short reason, and put missed items at the top of the next session. This keeps the next study block from starting with setup work.

## Checklist

- Before starting, define the **score section** output for today.
- Before ending, check **guessed item** and mark the next review item.
- Keep time spent, correct items, and missed items in one table.
- If the routine is too complex, remove one step and compare again next week.

## FAQ

### Should I apply Practice Test Review: Break Down Evidence Before Chasing Scores to every subject immediately?

Start with one subject, one unit, and one review cycle. Expand **Practice Test Review: Break Down Evidence Before Chasing Scores** only after the **score section** record is useful in the next session.

### Can this work when study time is short?

Yes, if the short session still checks **guessed item** and leaves a closing record. In **Practice Test Review: Break Down Evidence Before Chasing Scores**, time alone is not the point; retrieval, feedback, and rescheduling need to be included.

### Is Practice Test Review: Break Down Evidence Before Chasing Scores failing if scores do not improve immediately?

No. **Practice Test Review: Break Down Evidence Before Chasing Scores** first becomes valuable by revealing repeated failure points. Keep the same **score section** measure for two or three weeks before changing the system.
